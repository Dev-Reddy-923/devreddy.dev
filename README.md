# Dev Reddy Portfolio

Starter portfolio website built with Next.js and Tailwind CSS.

## Run Locally

```bash
pnpm install
pnpm dev
```

Visit [http://localhost:3000](http://localhost:3000).

## Edit Content

Update portfolio content in:

- `app/page.tsx` for hero, projects, and contact sections
- `app/layout.tsx` for SEO title and description

## Deploy to Vercel

1. Push this repo to GitHub.
2. Open [Vercel](https://vercel.com/new) and import the repository.
3. Keep the default Next.js build settings.
4. Click **Deploy**.

After deployment, every push to `main` will trigger a new production deploy.
